## 项目概述

**Metric Bot** 是一个智能运维分析平台，通过自然语言交互、智能告警和多数据源集成，为用户提供高效的运维分析能力。

**核心定位：**
- 🔍 **观察**：监控数据采集和可视化
- 🧠 **分析**：告警聚合、根因分析、趋势预测
- 💡 **建议**：智能诊断、最佳实践推荐
- 📊 **报告**：运维报告自动生成

> **注意**：本项目聚焦于"分析"而非"操作"，应急处置功能（如自动重启服务、执行扩缩容等）规划为独立项目。

## 模块规划

### 模块一：监控面板

**状态：** 🟡 进行中（前端页面已完成）

**功能描述：**
- 展示当前触发的告警数量
- 展示历史告警统计数据
- 提供告警趋势可视化图表
- 支持告警级别分类展示（严重、警告、信息）

**核心指标：**
- 实时告警数
- 今日告警数
- 本周告警数
- 告警处理率

**技术要点：**
- 实时数据推送（WebSocket）
- 图表可视化（ECharts）
- 数据聚合与缓存策略

## 技术架构

### 前端技术栈
- Vue 3 + TypeScript
- Element Plus UI 组件库
- Pinia 状态管理
- Vue Router 路由管理
- ECharts 图表库
- Axios HTTP 客户端

### 后端技术栈
- Python FastAPI
- SQLAlchemy ORM
- MySQL 数据库
- JWT 认证
- WebSocket 实时通信
