# Metric Bot 项目模块规划文档

> 最后更新时间：2026-03-15
> 项目状态：开发中

---

## 项目概述

**Metric Bot** 是一个智能运维监控平台，通过自然语言交互、智能告警和多数据源集成，为用户提供高效的运维分析能力。

---

## 模块规划

### 模块一：监控面板

**状态：** 🟡 进行中（前端页面已完成）

**功能描述：**
- 展示当前触发的告警数量
- 展示历史告警统计数据
- 提供告警趋势可视化图表
- 支持告警级别分类展示（严重、警告、信息）

**核心指标：**
- 实时告警数
- 今日告警数
- 本周告警数
- 告警处理率

**技术要点：**
- 实时数据推送（WebSocket）
- 图表可视化（ECharts）
- 数据聚合与缓存策略

---

### 模块二：智能对话

**状态：** 🟡 进行中（前端页面已完成）

**功能描述：**
- 通过自然语言进行运维分析
- 支持查询监控指标、日志、告警等数据
- 提供智能诊断建议
- 对话历史记录与回溯

**参考项目：** SQLBot（位于 `SQLBot/` 目录）

**核心功能：**
- 自然语言转查询语句
- 多轮对话上下文管理
- 查询结果可视化展示
- 对话历史导出

**技术要点：**
- 大模型集成（支持多模型切换）
- 上下文管理
- 查询结果渲染
- 流式响应处理

---

**功能描述：**

## 技术架构

### 前端技术栈
- Vue 3 + TypeScript
- Element Plus UI 组件库
- Pinia 状态管理
- Vue Router 路由管理
- ECharts 图表库
- Axios HTTP 客户端

### 后端技术栈
- Python FastAPI
- SQLAlchemy ORM
- MySQL 数据库
- JWT 认证
- WebSocket 实时通信
